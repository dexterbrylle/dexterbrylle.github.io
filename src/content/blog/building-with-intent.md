---
title: "Building with Intent"
description: "On the value of small, focused tools built for specific problems."
pubDate: 2026-05-09
tags: ["software", "philosophy"]
---

There's a difference between software that exists and software that solves.

The first category is everywhere: bloated, feature-creeped, trying to be everything
to everyone. It accumulates options and switches and configuration screens until
using it requires reading documentation.

The second is rarer. Small. Focused. It does one thing well. You understand it
immediately because there is nothing to misunderstand.

I try to build the second kind.
